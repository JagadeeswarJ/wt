# Task 03 — CSS Styling

Apply CSS styling to enhance visual appeal with fonts, colors, and layout.

## File
- `index.html` — full bookstore page with embedded CSS

## How to Run
1. Double-click `index.html` → opens in any browser (Chrome, Firefox, Edge)
2. No setup needed — pure HTML + CSS

## What's Styled
| Element | Style |
|---------|-------|
| Body font | `Georgia` serif |
| Primary color | `#2c3e50` (dark blue-grey) |
| Accent color | `#e74c3c` (red) |
| Book cards | Box shadow, hover effect |
| Layout | CSS Flexbox for card grid |
| Header/Nav | Fixed top bar with links |
